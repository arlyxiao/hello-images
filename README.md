# hello-images
Make image search easier.

### Requirements
1. Follow the node version in .nvmrc.

## Building and running on localhost

Prepare env variables with right value (for example: **API_KEY**):
```
cp .env.example .env.local
```

Installation
```sh
npm install
```

For Development
```sh
npm start
```

Create a production build:

```sh
npm run prod
```

## Testing

To run unit tests:

```sh
npm test
```
